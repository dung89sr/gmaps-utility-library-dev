KeyDragZoom Examples

Include Scripts

The first step is to include keydragzoom.js or keydragzoom_packed.js in your document header after the Google Maps API v2 has been included. You can use the online version if you do not want to download the script.

      <script src="/path/to/keydragzoom_packed.js" type="text/javascript"></script>

Enabling KeyDragZoom

You simply call GMap2.enableKeyDragZoom() to enable KeyDragZoom, just like you would call GMap2.enableScrollWheelZoom() to enable scrollwheel zooming.

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom();  
        }

Styling the Zoom Box

You can pass some css properties when you enable KeyDragZoom. They will be applied to the zoom box. In the following example, the zoom box is half transparent red with a thick dashed black border.

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom({
                boxStyle: {
                  border: "thick dashed black",
                  backgroundColor: "transparent",
                  opacity: 1.0
                },
                paneStyle: {
                  backgroundColor: "gray",
                  opacity: 0.25
                }
          });
        }

Using a Different Hot Key

If you do not want to use the SHIFT key as the hot key, you can pass one of the other two keys: ALT or CTRL. In the following example, you can start a drag zoom operation by holding down the CTRL key

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom({
            key: 'ctrl'
          });
        }

Visual Drag Zoom Control

You can also use a visible control to turn KeyDragZoom on and off. Simply supply the position of the control as well as the URLs for the pictures to show when the control in the on, off, and hot states. In this example the control is placed below the zoom control.

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom({
            imagePosn: new GPoint(27, 285),
            imageOn: "dragzoom-on.png",
            imageOff: "dragzoom-off.png",
            imageHot: "dragzoom-hot.png"
          });
        }

Multiple Maps Supported

Each map can have its own independent KeyDragZoom tool.

        function init(){
          var map1 = new GMap2(document.getElementById("map1"));
          map1.enableKeyDragZoom();
          var map2 = new GMap2(document.getElementById("map1"));
          map2.enableKeyDragZoom();
        }

KeyDragZoom Events

The KeyDragZoom tool triggers different events at key moments of the drag zoom operation.

       map.enableKeyDragZoom();
          var dz = map.getDragZoomObject();
          GEvent.addListener(dz, 'activate', function() {
            GLog.write('DragZoom Activated');
          });
          GEvent.addListener(dz, 'dragstart', function() {
            GLog.write('DragZoom Started');
          });
          GEvent.addListener(dz, 'drag', function() {
            GLog.write('DragZoom Dragging...');
          });
        }
