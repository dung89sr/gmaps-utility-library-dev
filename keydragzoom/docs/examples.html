Including Scripts

The first step is to include keydragzoom.js or keydragzoom_packed.js in your document header after the Google Maps API v2 has been included. You can use the online version if you do not want to download the script.

      <script src="/path/to/keydragzoom_packed.js" type="text/javascript"></script>

Enabling KeyDragZoom

You simply call GMap2.enableKeyDragZoom() to enable KeyDragZoom, just as you would call GMap2.enableScrollWheelZoom() to enable scrollwheel zooming.

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom();  
        }

Styling the Zoom Box and the Veil

You can pass some css properties when you enable KeyDragZoom. Those associated with boxStyle are applied to the zoom box the user drags across the area of interest and those associated with veilStyle are applied to the veil that covers the map when the hot key is down. In the following example, the zoom box is transparent with a 4-pixel dashed black border and the veil is red with 25% opacity.

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom({
                boxStyle: {
                  border: "4px dashed black",
                  backgroundColor: "transparent",
                  opacity: 1.0
                },
                veilStyle: {
                  backgroundColor: "red",
                  opacity: 0.25
                }
          });
        }

Using a Different Hot Key

If you do not want to use the Shift key as the hot key, you can pass one of the other two keys: Alt or Ctrl. In the following example, you can start a drag zoom operation by holding down the Ctrl key.

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom({
            key: 'ctrl'
          });
        }

Visual Drag Zoom Control

You can also use a visual control to turn KeyDragZoom on and off. Simply set the visualEnabled parameter to true, indicate the position of the control, and provide the URLs for the pictures to show when the control is in the on, off, and hot states. (The control is hot when the mouse moves over it.) In this example the control is placed below the standard zoom control.

        function init(){
          var map = new GMap2(document.getElementById("map"));
          map.setCenter(new GLatLng(45.5, -122.7), 14);
          //...
          map.enableKeyDragZoom({
            visualEnabled: true,
            visualPosition: new GControlPosition(G_ANCHOR_TOP_LEFT, new GSize(27, 285)),
            imageOn: "dragzoom-on.png",
            imageOff: "dragzoom-off.png",
            imageHot: "dragzoom-hot.png"
          });
        }

Multiple Maps Supported

Each map on the same webpage can have its own independent KeyDragZoom tool.

        function init(){
          var map1 = new GMap2(document.getElementById("map1"));
          map1.enableKeyDragZoom();
          var map2 = new GMap2(document.getElementById("map2"));
          map2.enableKeyDragZoom();
        }

KeyDragZoom Events

Different events are triggered at key moments of a drag zoom operation.

       map.enableKeyDragZoom();
       var dz = map.getDragZoomObject();
       GEvent.addListener(dz, 'activate', function() {
         GLog.write('KeyDragZoom Activated');
       });
       GEvent.addListener(dz, 'deactivate', function() {
         GLog.write('KeyDragZoom Deactivated');
       });
       GEvent.addListener(dz, 'dragstart', function(latlng) {
         GLog.write('KeyDragZoom Started: ') + latlng;
       });
       GEvent.addListener(dz, 'drag', function(startPt, endPt) {
         GLog.write('KeyDragZoom Dragging: ') + startPt + endPt;
       });
       GEvent.addListener(dz, 'dragend', function(bnds) {
         GLog.write('KeyDragZoom Ended: ') + bnds;
       });
