SnapToRoute

Class to make a marker connect to a polyline. Also calculates the distance along the line the snapped maker is. You can update the targets of the class through a method.

Based on Distance to line example by Marcelo, maps.forum.nu. Then @ work of Björn Brala - Swis BV who wrapped the algorithm in a class operating on GMap Objects. And Bill Chadwick who factored the basic algorithm out of the class (removing much intermediate storage of results) and added distance along line to nearest point calculation
