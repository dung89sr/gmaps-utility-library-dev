PopupMarker

This library gives the popup next to with a marker. It has two method. One is simple popup. The popup can show texts, images and HTML, but it can not change background-color. Another popup is using output of the Google Chart API. It can show texts and changees the background-color by some options. But it NOT support a lot of change. (e.g. change texts when on dragging event) You can select it by set the "style" option when creating new the PopupMarker. ref. The Google Chart API - Chart types for information bubbles

For a description and examples of how to use this library, check out the how-to.

class PopupMarker

Constructor

Constructor Description
PopupMarker(latlng:GLatLng, opts?:PopupMarkerOptions) Creates a marker with options specified in PopupMarkerOptions (extension of GMarkerOptions). Creates a popup and then calls the GMarker constructor.

Methods

Methods Return Value Description
hide() None Hides the marker and popup
hidePopup() None Hides the popup
setChartBgColor(bgColor:String) None Set background color, also as a 6-digit hexadecimal number. If the style property not set "chart", then this property is ignored.
setChartIcon(iconName:String) None Set icon's name of the Google Chart API. If the style property not set "chart", then this property is ignored.
setChartStyle(styleName:String) None Set chart style for the Google Chart API. If the style property not set "chart", then this property is ignored.
setChartTextColor(textColor:String) None Set text color, also as a 6-digit hexadecimal number. If the style property not set "chart", then this property is ignored.
setShapeStyle(colorValue:String) None Set shape style for the Google Chart API. This may, in the future, allow you to use different shapes for the bubble. For now, "bb" is the only valid option.
setText(message:Strings) None Set the text of the popup message.
show() None Shows marker. Note that this method shows only the marker. If you want show marker and the popup, then use the showPopup method.
showPopup() None Shows the marker and the popup.

class PopupMarkerChartAPIOptions

This call represents options passed within the PopupMarkerChartAPIOptions to the PopupMarkerOptions object. It has no constructor, but is instantiated as an object literal. There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
bgColor String Specifies background color, also as a 6-digit hexadecimal number. The default value is "FFFFFF".
chartStyle String Specifies style for the Google Chart API, chooses from "d_bubble_icon_text_small", "d_bubble_icon_text_big", "d_bubble_icon_texts_big" or "d_bubble_texts_big". The default value is "".
icon String Specifies, icon's name of the Google Chart API. The default value is "".
shapeStyle String Specifies shape style for the Google Chart API. This may, in the future, allow you to use different shapes for the bubble. For now, "bb" is the only valid option. The default value is "bb".
textColor String Specifies text color, as a 6-digit hexadecimal number, for example 000000 for black, FF0000 for red, FFFFFF for white, or FFFF00 for yellow. The default value is "000000".

class PopupMarkerOptions

This class represents optional arguments to PopupMarker and GMarker. Each of the functions use a subset of these arguments. See the function descriptions for the list of supported options. It has no constructor, but is instantiated as an object literal. There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
chart PopupMarkerChartAPIOptions This property specifies the options to configure the Google Chart API. These options are passed to the PopupMarkerOptions object literal when the marker is constructed, and are used to construct the PopupMarker when PopupMarker.showPopup() is called. If the style property not set "chart", then this property is ignored. The default value is {}.
style String Specifies style for popup, chooses from "normal" or "chart". If it set "chart", then this library shows the popup using output of the Google Chart API. The default value is "normal".
text String Specifies text for popup. If the style property set "chart", then it follow format for text property of the Google Chart API. The default value is "".
