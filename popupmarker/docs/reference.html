PopupMarker

Creates a popup with marker.

For a description and examples of how to use this library, check out the how-to.

class PopupMarker

Constructor

Constructor Description
PopupMarker() Creates a PopupMarker at the latlng with options specified in PopupMarkerOptions and GMarkerOptions. By default markers are clickable & have the default icon G_DEFAULT_ICON.

Methods

Method Return
Value		 Description
setTitle None Set marker's title.Able to HTML text.
getTitle Strings Return marker's strings.
showPopup None Show popup.
hidePopup None Hide popup.

class PopupMarkerOptions

Properties

Property Type Description
style String The "normal" is default popup.
The "chart" is the Charts API.
