PopupMarker Example

To create a popup marker, create a new PopupMarker, and specify the relevant options. The most important options are style and text. Then you can assign it event listeners and add it to the map just as you would with a GMarker object.

var map = new GMap2(document.getElementById("map_canvas"));
map.setCenter(new GLatLng(35.681382, 139.766084), 1);

var opts = { text : "Hello! I'm a PopupMarker."};

var marker = new PopupMarker(new GLatLng(35.681382, 139.766084), opts);
map.addOverlay(marker);
