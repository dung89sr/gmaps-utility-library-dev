PopupMarker Example

To use the progress bar, create a PopupMarker object and add this marker to the map with GMap2.addOverlay(PopupMarker).

var map = new GMap2(document.getElementById("map_canvas"));
map.setCenter(new GLatLng(48.859862, 2.292024), 17);
var markerOpt = {title:"PopupMarker example01"};
var marker = new PopupMarker(new GLatLng(48.859862, 2.292024),markerOpt);
map.addOverlay(marker);
marker.showPopup();
