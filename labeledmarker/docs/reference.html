LabeledMarker

This library extends the Maps API's standard GMarker class with the ability to support markers with textual labels. Please see articles here:

  • http://googlemapsbook.com/2007/01/22/extending-gmarker/
  • http://googlemapsbook.com/2007/03/06/clickable-labeledmarker/

For a description and examples of how to use this library, check out the how-to.

class LabeledMarker

Constructor

Constructor Description
LabeledMarker(latlng:GLatLng, opt_opts?:LabeledMarkerOptions) Creates a marker with options specified in LabeledMarkerOptions (extension of GMarkerOptions). Creates a div for the label and then calls the GMarker constructor.

Methods

Methods Return Value Description
copy() None Return a copy of this overlay, for the parent GMap2 to duplicate itself in full. This is part of the GOverlay interface and is used, for example, to copy everything in the main view into a mini-map.
getLabelVisibility() Boolean Returns whether label visibility is set on.
hide() None Hides the marker and the label if they are currently visible. Note that this function triggers the GMarker.visibilitychanged event in case the marker is currently visible.
hideLabel() None Hides the label of the marker. Note that this does not persist after a marker hide/show.
initialize(map:GMap2) None Is called by GMap2's addOverlay() method. Adds the text div to the relevant parent div.
redraw(force:Boolean) None Calls the redraw() handler in GMarker and our redrawLabel_() function.
remove() None Remove the text div from the map pane, destroy event passthrus, and calls the default remove() handler in GMarker.
setLabelText(text:String) None Sets label text/html for the marker.
setLabelVisibility(visibility:Boolean) None Sets the visibility of the label, which will be respected during show/hides. If marker is visible when set, it will show or hide label appropriately.
setLatLng(latlng:GLatLng) None Repositions label and marker.
show() None Shows the marker and shows the label if it wasn't hidden. Note that this function triggers the GMarker.visibilitychanged event in case the marker is currently hidden.
showLabel() None Shows the label of the marker. Note that this does not persist after a marker hide/show.

class LabeledMarkerOptions

This class extends GMarkerOptions. Instances of this class are used in the opt_opts argument to the constructor of the LabeledMarker class. At this time the draggable property is not supported. There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
labelClass String Class to use for the overlay div (default "LabeledMarker_markerLabel").
labelOffset GSize Label offset, the x- and y-distance between the marker's latlng and the upper-left corner of the text div (default (0,0)).
labelText String Label text/html to place in the overlay div (defaults to empty string).
