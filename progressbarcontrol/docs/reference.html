ProgressbarControl

Creates a progressbar control for usage in google maps.

Usage:
progressbar = new ProgressbarControl(oMap, opt_opts);
progressbar.(500); // Amount of operations, unhides the control
progressbar.(iAdded); // Add amount of operations just done
progressbar.(); // Hide the control.

opt_opts: Object containing options:
{Number} [width=176] Width of the control
{String} [loadstring=Loading...] String displayed when first starting the control

class ProgressbarControl

Methods

Methods Return Value Description
remove() None Remove control. Well, hide it actually, since the call to create a new one when its needed again would create to much overhead.
start(operations) None Start the progress bar. Argumnent is the amount of operations the full bar will represent.
updateLoader(step) None Update the progress. Adds Step amount of operations to the bar.

class ProgressbarOptions

This class represents optional arguments to ProgressbarControl, There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
loadstring String Specifies the string displayed when first starting the control. Before any update! The default value is Loading....
width Number Specifies, in pixels, the width of the progress bar. The default value is 176.
