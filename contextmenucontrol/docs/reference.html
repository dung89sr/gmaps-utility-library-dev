class ContextMenuControl

This class lets you add a context menu to the map which mimics the context menu of Google Maps. All currently available six items of the context menu will be shown by default, i.e. when no options are specified. By specifying additional options any particular item can be excluded. This control extends GControl.

Note: ContextMenuControl doesn't work in Opera because Opera doesn't support the oncontextmenu event, and right mouse clicks aren't accessible with Javascript - neither mousedowns nor mouseups.

Constructor

Constructor Description
ContextMenuControl(opt_options) Creates a ContextMenuControl with the specified option hash (optional).

class ContextMenuControlOptions

Instances of this class are used in the opts_options argument to the constructor of the ContextMenuControl class. There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Optional Properties

Each property is optional and - if used - should be severally set to false. The value true causes the same result as if no particular option was specified, i.e. the item will be shown.

Properties Value Description
dirsFrom false If set to false, ContextMenuControl's item 'Directions from here' will not be shown.
dirsTo false If set to false, ContextMenuControl's item 'Directions to here' will not be shown.
zoomIn false If set to false, ContextMenuControl's item 'Zoom In' will not be shown.
zoomOut false If set to false, ContextMenuControl's item 'Zoom Out' will not be shown.
centerMap false If set to false, ContextMenuControl's item 'Center Map here' will not be shown.
whatsHere false If set to false, ContextMenuControl's item 'What's here?' will not be shown.
