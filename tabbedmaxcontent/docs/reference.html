Tabbed Max Content

This lib provides a similar infowindow behavior avaiable at maps.google.com. The maxContent of info window contains multiple tabs.

For a description and examples of how to use this library, check out the how-to.

class GMap2

These are new methods added to Google Maps API's GMap2 class.

Methods

Methods Return Value Description

Each tab is an instance of MaxContentTab, The content of each tab is html. opt_maxOptions is an instance of MaxContentOptions.

class MaxContentOptions

Instances of this class are used in the opts_maxOption argument to methods openMaxConentTabs(), openMaxContentTabsHtml(). It will eventually be passed to GMap2.openInfoWindow as GInfoWindowOptions. Note, GInfoWindowOptions.maxContent can not be used. For other properties, see GInfoWindowOptions There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
maximized Boolean Specifies if the window should be open as maximized by default.
maxTitle String Specifies title to be shown when the infowindow is maximized. The content may be either an HTML string or an HTML DOM element.
selectedTab Number|String Selects the tab with the given index or name. index base is 0.
style Object the object that hold a set of css style of the max content. It has the following properties: tabOn, tabOff, tabBar, tabLeft, tabRight, content . Each property is a css object such as {backgroundColor: 'gray', opacity: 0.2}.

class MaxContentTab

This class represents a tab in the max content window. An array of instances of this class can be passed in as the tabs argument to the methods openMaxContentTabs*() etc. This class is similar to the GInfoWindowTab class in core API.

Constructor

Constructor Description
MaxContentTab(label:String, content:Node|String) Creates a content tab data structure that can be passed in the tabs argument in the openMaxContentTabs*() methods.

Methods

Methods Return Value Description
getContentNode() Node Get the content DOM Node of the tab
getLabel() String Get the label of the tab

class TabbedMaxContent

This class represent the max content in the info window. It has three parts: summary info; tab Navigation bar, and tabbed contents. There is no public constructor for this class. If needed, it can be accessed via GMap2.getInfoWindowMaxContent()
