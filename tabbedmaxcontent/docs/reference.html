Tabbed Max Content

This lib provides a similar infowindow behavior avaiable at maps.google.com. The maxContent of info window contains multiple tabs.

For a description and examples of how to use this library, check out the how-to.

class GMap2

These are new methods added to Google Maps API's GMap2 class.

Methods

Methods Return Value Description
getInfoWindowMaxContent() TabbedMaxContent Returns the TabbedMaxContent for the infowindow.
openInfoWindowMaxTabs(latlng:GLatLng, node:Node, summary:Node, tabs:GInfoWindowTab[], opt_maxOptions:MaxInfoWindowOptions) None Opens an info window with max content at the given point. The content of the info window is given as DOM node. The max content has summary info and a set of tabs. The content of each tab is DOM node.
openInfoWindowMaxTabsHtml(latlng:GLatLng, html:String, summary:String, tabs:GInfoWindowTab[], opt_maxOptions:MaxInfoWindowOptions) None Opens an info window with max content at the given point. The content of the info window is given as HTML text. The max content has summary info and a set of tabs. The content of each tab is html.

class MaxInfoWindowOptions

Instances of this class are used in the opts_maxOption argument to the methods openInfoWindowMaxTabs(), openInfoWindowMaxTabsHtml(). It's basically same as GInfoWindowOptions and will actually be used to pass in as GInfoWindowOptions to openInfoWindow Please note the GInfoWindowOptions.maxContent is not applicable. For other properties, see GInfoWindowOptions There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
maxTitle String Specifies title to be shown when the infowindow is maximized. The content may be either an HTML string or an HTML DOM element.
selectedTab Number|String Selects the tab with the given index or name. index base is 0.
style Object the object that hold a set of css style of the max content. It has the following properties: tabOn, tabOff, tabBar, tabLeft, tabRight, content . Each property is a css object such as {backgroundColor: 'gray', opacity: 0.2}.

class TabbedMaxContent

This class represent the max content in the info window. It has three parts: summary info; tab Navigation bar, and tabbed contents. There is no public constructor for this class. If needed, it can be accessed via GMap2.getInfoWindowMaxContent()
