SnapShotControl Examples

To use the progress bar, create a SnapShotControl object and add this control to the map with GMap2.addControl(SnapShotControl). 

var map = new GMap2(document.getElementById("map_canvas"));
map.setCenter(new GLatLng(35.676148, 139.74479), 1);

// add control
var snapShotControl = new SnapShotControl();
map.addControl(snapShotControl);

Basic Example

The following example shows the SnapShotControl in a simple example.

var map = new GMap2(document.getElementById("map_canvas"));
map.setCenter(new GLatLng(35.676148, 139.74479), 1);

// add control
var snapShotControl = new SnapShotControl();
map.addControl(snapShotControl);

// add maptype control
var menuMapTypeControl = new GMenuMapTypeControl();
map.addControl(menuMapTypeControl);

// add maptype
map.addMapType(G_PHYSICAL_MAP);

// add line
var points=[];
points.push(new GLatLng(35.676148, 139.74479));
points.push(new GLatLng(25.48295117535531, 118.30078125));
points.push(new GLatLng(-32.39851580247401, 135.17578125));
points.push(new GLatLng(37.43997405227057, -121.9921875));
var line = new GPolyline(points, "#FF0000", 10);
map.addOverlay(line);

// add markers
for (var i = 0; i < points.length; i++) {
  var marker = createMarker(points[i]);
  map.addOverlay(marker);
}
