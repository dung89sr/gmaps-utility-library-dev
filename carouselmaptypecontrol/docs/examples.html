CarouselMapTypeControl Example

To use the progress bar, create a CarouselMapTypeControl object and add this control to the map wiith GMap2.addControl(CarouselMapTypeControl).

var map = new GMap2(document.getElementById("map_canvas"));
map.setCenter(new GLatLng(37.4419, -122.1419), 1);
        
//add map types
map.addMapType(G_PHYSICAL_MAP);
map.addMapType(G_MOON_VISIBLE_MAP);

//add control
var carouselMapTypeControl = new CarouselMapTypeControl();
map.addControl(carouselMapTypeControl);
